Occupancy List

SAFHA - Hotel Manager

Occupancy List Page​

The Occupancy List page is designed to provide a comprehensive overview of current guest reservations and room occupancy. This section is essential for tracking and managing guest stays, ensuring that all reservations are accurately recorded and easily accessible.

Filters​

At the top of the page, there are filters to help narrow down the occupancy list:

  • Reservation Ref: A field to enter the reservation reference number.
  • Arrival Date From: A date picker to select the start date for the reservations you want to view.
  • Arrival Date To: A date picker to select the end date for the reservations you want to view.
  • Submit Button: Apply the selected filters to update the list of reservations.

Table of Occupancy​

The main section of the page displays a table with the following columns:

  • S/N: Serial number of the reservation.
  • Ref: The reservation reference number.
  • Details: Additional details about the reservation.
  • Room: The room number or category associated with the reservation.
  • Guest: The name of the guest.
  • Phone Number: The contact number of the guest.
  • Rate Plan Amount: The amount charged according to the rate plan.
  • Total: The total amount for the reservation.

Navigation​

At the bottom of the table, there are navigation options to move between pages of occupancy records, indicating that multiple pages of reservations may exist within the system.

Actions​

  • View Details: Click on the view icon to see more details about the reservation.
  • Edit: Click on the edit icon to modify the reservation details.
  • Delete: Click on the delete icon to remove the reservation from the list.

Export​

Users can export the list of occupancy records to an Excel file by clicking the "Export Excel" button.

This page is crucial for maintaining an organized and efficient record of all guest reservations and room occupancy, ensuring smooth operations and excellent guest service.

Chart Analysis Page

The Chart Analysis page provides a visual representation of data related to room occupancy, inventory, or other key metrics over a specified period. This page helps users quickly understand trends and patterns, making it easier to make informed decisions.

Bar Chart​

The main section of the page features a bar chart with the following components:

  • Title: The chart is titled "BAR CHART".
  • X-Axis: Labeled "Rooms", representing different rooms in the hotel. Each room is depicted with an icon of a bed.
  • Y-Axis: Labeled "Nº Items", indicating the number of items or occurrences being measured. The scale ranges from 0 to 1 in increments of 0.1.
  • Legend: Displays the date range for the data being analyzed, which in this example is from "August 2, 2024 to September 1, 2024".

Interpretation​

  • Empty Chart: If the chart is empty (no bars displayed), it indicates that there is no data available for the selected date range or criteria.
  • Data Representation: Each bar represents the number of items or occurrences for a specific room. Higher bars indicate higher counts.

Actions​

  • Filter Data: Users can adjust the date range or other criteria to filter the data displayed on the chart.
  • Export: Users can export the chart data to an Excel file for further analysis.

This page is essential for visualizing key metrics and trends, helping users make data-driven decisions to improve hotel operations.

Room Status (Next 30 Days)

The Room Status (Next 30 Days) section provides a comprehensive overview of room occupancy and availability for the upcoming month. This feature is essential for managing hotel capacity, ensuring accurate guest information, and optimizing room allocation.

Key Features​

  • Detailed Room List: Displays all rooms with their current status, including occupied, available, and reserved.
  • Guest Information: Shows guest names, check-in, and check-out dates for occupied rooms.
  • Date Range: Covers the next 30 days from the current date, allowing staff to plan and manage future reservations effectively.
  • Room Numbers: Lists room numbers for easy identification and management.

How to Use​

  1. View Room Status: Navigate to the "Room Status (Next 30 Days)" section to see a detailed list of rooms and their statuses.
  2. Monitor Occupancy: Check which rooms are occupied, available, or reserved to manage hotel capacity.
  3. Manage Reservations: Use the information to plan for upcoming check-ins and check-outs, ensuring smooth transitions for guests.
  4. Update Information: Regularly update guest information and room statuses to maintain accuracy.

Benefits​

  • Efficient Management: Helps staff efficiently manage room availability and guest information.
  • Improved Planning: Provides a clear view of room occupancy for the next 30 days, aiding in better planning and resource allocation.
  • Enhanced Guest Experience: Ensures that guest information is up-to-date, leading to a smoother check-in and check-out process.
