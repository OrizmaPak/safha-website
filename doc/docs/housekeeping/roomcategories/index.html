Room Categories

The Room Categories page allows you to manage different types of rooms available in your hotel. This helps in organizing and categorizing rooms based on their features, pricing, and other criteria.

Fields Explained:​

  • Category Name: The unique name assigned to each room category. For example, "Deluxe Room" or "Standard Room".
  • Category Type: A dropdown menu to select the type of room category, such as "Suite", "Single", or "Double".
  • Rate Code: A specific code that corresponds to the pricing structure for the room category.
  • Currency: The currency in which the room rates are displayed and calculated.
  • Minimum Deposit Required: The minimum amount that needs to be deposited to secure a booking for this room category.
  • Price: The cost associated with booking a room under this category. There are fields for entering different price levels, which can be used for various pricing strategies.

Example:​

To add a new room category, fill in the following details:

  1. Category Name: Enter the name of the room category (e.g., "Executive Suite").
  2. Category Type: Select the type of room from the dropdown menu (e.g., "Suite").
  3. Rate Code: Enter the rate code associated with this category (e.g., "EXS123").
  4. Currency: Choose the currency from the dropdown menu (e.g., "USD").
  5. Minimum Deposit Required: Enter the minimum deposit amount (e.g., "100").
  6. Price: Enter the price for the room category (e.g., "200" for the first price level and "250" for the second price level).

After filling in all the details, click the "Add" button to save the new room category.

By managing room categories effectively, you can ensure that your hotel's room offerings are well-organized and easily accessible for both staff and guests.
