Room Cleaning Checklist

SAFHA - Hotel Manager

View

This view allows supervisors to manage and track the status of room maintenance or inspections. Below is a detailed explanation of each column and its functionality:

Columns​

  • ID: A unique identifier for each task.
  • SUPERVISOR: The name of the supervisor responsible for the task.
  • ROOM NUMBER: The room number where the task is to be performed.
  • ITEMS: The items or tasks that need to be checked or completed. (Note: There seems to be a display error with "[object Object]" appearing here.)
  • ENTRY DATE: The date and time when the task was entered into the system.
  • SHIFT: Indicates whether the task is assigned to the day shift or another shift.
  • ACTION: Icons for actions such as edit (pencil), delete (trash can), and more options (three vertical dots).

Actions​

  • Edit: Click the pencil icon to edit the task details.
  • Delete: Click the trash can icon to delete the task.
  • More Options: Click the three vertical dots for additional options.

Notes​

  • Ensure that the "ITEMS" column displays the correct information instead of "[object Object]".
  • Regularly update the entry dates and shifts to keep the checklist current.

This view helps in maintaining the cleanliness and readiness of rooms by providing a clear and organized way to track tasks.

Update Checklist

This view allows supervisors to update and manage the checklist for room maintenance or inspections. Below is a detailed explanation of each section and its functionality:

Fields​

  • Supervisor: Enter the name of the supervisor responsible for the task.
  • Room Number: Enter the room number where the task is to be performed.
  • Entry Date: Select the date when the task was entered into the system.
  • Shift: Select the shift (e.g., Day Shift) during which the task is to be performed.

Tasks​

The following tasks can be checked off as they are completed:

  • Towel
  • Entry Door Cleaned
  • Home Cleaning
  • Water Bath
  • Washing Plates
  • Sweeping Room
  • Kitchen
  • Dusting
  • Clothes Selecting
  • Water

Actions​

  • Submit: Click the 'Submit' button to save the updates to the checklist.
