Build Page​

The Build feature allows users to create and manage inventory or orders efficiently. This feature is essential for maintaining accurate records and ensuring that all items are accounted for.

How to Use the Build Page Feature​

  1. Select Department/Sales Point:

    • Use the dropdown menu labeled Department/Sales Point to select the appropriate department or sales point.

  2. Set Build Date:

    • Use the date picker to select the Build Date. The current date is set to 09/02/2024 by default.

  3. Add Items:

    • In the ADD ITEM section, use the dropdown menu to Select Item.
    • Enter the Quantity of Item in the adjacent field.
    • Click the Add button to add the item to the list.

  4. Manage Items:

    • In the MANAGE ITEM section, a table displays the added items with the following columns:
      • S/N: Serial number of the item.
      • ITEM ID: Unique identifier for the item.
      • ITEM NAME: Name of the item.
      • QUANTITY: Quantity of the item.
      • ACTION: Options to edit or delete the item.

  5. Submit:

    • After adding and managing all items, click the Submit button to finalize the build.
